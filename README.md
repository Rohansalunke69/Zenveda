# Zenveda
Web-based preventive healthcare platform providing personalized Ayurvedic wellness reports

Zenveda is a web-based preventive healthcare platform that combines
Ayurvedic principles with modern technology to provide personalized
wellness guidance.

## Problem Statement
People often ignore early health symptoms due to lack of personalized
and preventive healthcare guidance. Ayurvedic knowledge, although
effective, is not easily accessible or understandable for modern users.

## Solution
Zenveda addresses this gap by allowing users to upload facial images
and enter health and lifestyle details. The system analyzes this data
to generate a personalized Ayurvedic wellness report focused on
prevention and lifestyle improvement.

## Key Features
- Web-based platform (no app installation required)
- Personalized Ayurvedic wellness reports
- Dosha analysis (Vata, Pitta, Kapha)
- Preventive health guidance
- Simple and user-friendly interface

## Technology Stack
- HTML, CSS, JavaScript
- Firebase (Authentication & Database)
- Google Cloud APIs (Image processing)
- AI-based logic for dosha evaluation

## How It Works
1. User opens the Zenveda website
2. Uploads facial image and enters symptoms
3. System processes inputs
4. Personalized Ayurvedic report is generated
5. User receives preventive lifestyle guidance

## Ethical Disclaimer
Zenveda does not diagnose diseases or prescribe medication.
It acts as a preventive wellness guidance system and encourages
professional medical consultation when required.

## Future Scope
- Multi-language support
- Continuous wellness tracking
- Practitioner assist mode
- Integration with wearable health data
